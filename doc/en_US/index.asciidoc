= Management of the shutters

== Description
Ce plugin permet de gérer facilement vos volets.
Celui-ci est entièrement basé sur le plugin Heliotrope qui est un prérequis pour l'utilisation de ce plugin.

Vous avez la possibilité de fermer vos volets à la tombée de la nuit et de les réouvrir au lever du soleil.
Vous avez également la possibilité de gérer vos volets pour qu'ils se ferment si le soleil est dans l'azimuth de votre fenêtre.
